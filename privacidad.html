Privacidad

Última actualización: 25 de junio de 2026.

Archivos y generación del PDF

Los archivos que el usuario selecciona se procesan en el navegador para crear el PDF. La aplicación utiliza esos archivos para armar la vista previa y la descarga limpia solicitada por el usuario.

License keys y créditos

Para validar una compra, PruebaPDF envía la license key ingresada a Lemon Squeezy y guarda un identificador técnico seguro junto con el saldo de créditos restante.

Datos de pago

Los datos de pago se gestionan en Lemon Squeezy. PruebaPDF usa los enlaces de checkout de Lemon Squeezy para cobrar y entregar license keys.

Almacenamiento local

El navegador puede guardar una sesión técnica para mostrar los créditos disponibles con mayor comodidad. El saldo real se consulta y descuenta del lado servidor.

Contacto

Para consultas de privacidad o soporte, escribí al correo de contacto configurado en la tienda.
